## 🚀 Proyectos destacados

- **Lab 01 · Landing Pastelería Maná** — Sitio responsivo real para una pastelería de Río Cuarto, Alajuela. Incluye modo oscuro, controles de tamaño de fuente, skip links y video de fondo en el hero, siguiendo WCAG 
- **lab02-interfaces-dom · Tour Virtual de Sedes (Mundial 2026)** — Aplicación JavaScript interactiva que consume la API REST pública del Mundial 2026 (`/get/stadiums`, `/get/games`) con autenticación JWT y `async/await` estricto. Permite navegar entre las 16 sedes mediante `scrollIntoView` suave y estado visual de "sede activa", filtrando los partidos por sede. Implementa arquitectura de resiliencia completa: manejo de error 401 sin recargar la página, backoff exponencial con countdown ante 429/500, y modo offline con `localStorage`.
